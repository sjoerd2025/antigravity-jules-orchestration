## 🔍 TROUBLESHOOTING

### If MCP servers don't start:

**Check logs:**
```
Tools → MCP → View Logs
```

**Look for:**
- "Connection state: Running" ✅ Good
- "Connection state: Stopped" ❌ Problem

**Common fixes:**
1. Wait full 30 seconds for initialization
2. Restart IntelliJ again
3. Check Docker is still running
4. Verify ChromaDB is on port 8000

### If auto-diagnostics fails:

Run quick-check manually:
```powershell
.\scripts\quick-check.ps1
```

**It will tell you exactly what's wrong:**
- Docker not running?
- ChromaDB not accessible?
- Jules server down?
