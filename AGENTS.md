# Autonomous AI Orchestration Agents

## Antigravity Agent
Primary coding agent with MCP integration for tool access. Handles code generation, debugging, and repository operations.

**Input:** Natural language development requests
**Output:** Code changes, PRs, file modifications

## Jules API Orchestrator
Programmatic session manager that coordinates autonomous development workflows.

**Input:** API requests with repository context and tasks
**Output:** Automated development sessions with plan approval

## Integration Flow
Antigravity ↔ MCP Tools ↔ Jules API ↔ GitHub Actions ↔ Render Deployment
